# SFD Comms
SFD Comms is a common Single Front Door (SFD) component that allows services to send communications to users using GOV.UK Notify.

## API
This component is a asynchronous API that accepts requests for communication notifications via Azure Service Bus. This has been documented in AsyncAPI which is available in the below table:

| Version | GitHub | Status |
|---------|-------|--------|
| [3.3.0](v3.md) | [asyncapi.yml](https://github.com/DEFRA/fcp-fd-comms/blob/main/docs/asyncapi.yaml) | Latest |
